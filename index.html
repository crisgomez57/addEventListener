mouse

mousedown: El usuario presiona el botón del ratón en un elemento (aporreado)

mousemove: El usuario mueve el puntero del ratón sobre un elemento (cosquillas)

mouseout: El usuario mueve el puntero fuera de un elemento (triste)

mouseup: El usuario libera el botón pulsado del ratón sobre un elemento (sorpresa)

mouseover No choques con las tuberías
